    • M(Model) 模型用于封装数据及数据处理方法
    • V(View) 视图实时更新数据，包括事件注册
    • C(ontroller) 控制器接收用户的操作，最主要是订阅视图层的事件
    • Event 发布订阅模式，让多个观察者同时监听，当一个事件发生改变，所有依赖都将得到通知
